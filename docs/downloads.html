Data sources

All the data used in this project was downloaded from public databases. Expression data and sample metadata was downloaded from the PODC website. Orthology, gene trees and gene metadata from http://plants.ensembl.org (mostly via biomart API). As some of the expression data from PODC used a different genome annotation the corresponding expression data was downloaded using EBI’s RNAseq API.

How to download everything

Everything should be automatically downloaded by running the source of this document (downloads.Rmd) in R.

Fix the metadata

Because not all of the samples described in the metadata could be downloaded from EBI, the corresponding entries should be removed from the metadata to keep it consistant. Furthermore, the order of the samples in the metadata should be ordered to match the order in the PODC expression data.

  • Gm : removed 12 sample metadata entries.
  • Zm : removed 25 sample metadata entries.
